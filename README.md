Homepage for Apollo Hub built with Next.js and TypeScript.

## Getting started

```bash
npm install
npm run dev
```

Then open http://localhost:3000 in your browser.

## Requirements

- Node.js 20.9 or newer
- npm, pnpm, or yarn
